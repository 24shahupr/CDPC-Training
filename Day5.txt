

## PYTHON BASIC PROGRAMS (HACKERRANK PRACTICE)

1. Print “Hello World” in Python

---

This program simply displays the message **Hello, World!** on the screen.

print("Hello, World!")

---

2. Python If–Else Condition

---

This program checks the value of a number and prints different messages depending on the condition.

num = int(input())

if num % 2 != 0:
print("Weird")

elif num >= 2 and num <= 5:
print("Not Weird")

elif num >= 6 and num <= 20:
print("Weird")

else:
print("Not Weird")

---

3. Arithmetic Operations

---

This program performs basic arithmetic calculations on two integers.

a = int(input())
b = int(input())

print(a + b)
print(a - b)
print(a * b)

---

4. Division Operations

---

This program demonstrates two types of division in Python.

a = int(input())
b = int(input())

print(a // b)
print(a / b)

Here:
// → integer (floor) division
/ → normal floating-point division

---

5. Loop Example

---

This program prints the square of numbers from **0 up to n-1**.

n = int(input())

for i in range(n):
print(i * i)
